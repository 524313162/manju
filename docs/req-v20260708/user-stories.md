# ManjuCraft.Web AI 创作能力对接 — 用户故事文档

> 版本：1.0 | 日期：2026-07-08 | 关联需求文档：requirements.md

## 用户角色

| 角色     | 描述                                 |
| -------- | ------------------------------------ |
| 编剧     | 负责剧本创作、章节编辑、改写、扩写   |
| 导演     | 负责分镜脚本拆分、分镜图确认、视频生成 |
| 美术     | 负责角色图、场景图的设计与审核       |
| 音频     | 负责 BGM 音乐创作与音频配置          |

---

## Epic 1：故事创作

### US-001：AI 一键生成剧本

**优先级**：Must Have

作为编剧，
我希望输入故事主题后点击"AI 创作"自动生成剧本章节，
以便快速获得完整的故事结构。

**验收标准**：
- [ ] 用户可以输入故事标题和主题描述
- [ ] 用户可以查看/编辑提示词模板
- [ ] 点击"AI 创作"后显示"生成中..."状态
- [ ] 生成完成后自动解析 JSON 并保存为章节
- [ ] 每个章节按顺序显示在左侧章节列表中
- [ ] 失败时显示具体错误信息

### US-002：AI 改写/扩写章节

**优先级**：Must Have

作为编剧，
我希望对已有章节进行 AI 改写、扩写、润色或精简，
以便在不手动编辑的情况下优化章节内容。

**验收标准**：
- [ ] 每个章节有"AI 改写/扩写"按钮
- [ ] 用户可修改原文内容
- [ ] 用户可编辑提示词模板
- [ ] 点击开始后显示"改写中..."状态
- [ ] AI 返回改写结果后，可在弹窗中预览
- [ ] 确认后可直接保存更新章节

## Epic 2：资产管理（AI 生成）

### US-003：AI 生成角色图片

**优先级**：Must Have

作为美术，
我希望输入角色描述后点击"AI 生成"自动生成角色图片并自动替换到资产卡片，
以便快速获得角色视觉形象。

**验收标准**：
- [ ] 角色卡片上有"AI 生成"按钮
- [ ] 点击后弹出输入窗口（可输入生成参数如尺寸）
- [ ] 显示"生成中..."loading 状态
- [ ] 成功后图片自动替换到角色卡片
- [ ] 超时（10 分钟）后显示 promptId，可手动"获取资产"
- [ ] 可手动指定图片尺寸

### US-004：AI 生成场景图片

**优先级**：Must Have

作为美术，
我希望输入场景描述后点击"AI 生成"自动生成场景图片并自动替换到资产卡片，
以便快速获得场景参考图。

**验收标准**：
- [ ] 场景卡片有"AI 生成"按钮，交互同 US-003
- [ ] 成功后图片自动替换到场景卡片
- [ ] 超时可手动获取资产

### US-005：AI 生成 BGM 音乐

**优先级**：Must Have

作为音频，
我希望输入音乐描述后点击"AI 生成"自动生成音乐并自动替换到资产卡片，
以便快速获得配乐。

**验收标准**：
- [ ] BGM 卡片有"AI 生成"按钮
- [ ] 点击后显示"生成中..."状态
- [ ] 成功后音频自动替换到 BGM 卡片
- [ ] 音频卡片支持播放/暂停
- [ ] 超时可手动获取资产

### US-006：AI 生成角色配音（占位）

**优先级**：Must Have

作为音频，
我希望输入文本后点击"AI 配音"生成角色配音音频，
以便为角色赋予声音形象。

**验收标准**：
- [ ] 角色卡片有"AI 配音"按钮
- [ ] 当前为占位功能，TTS 接口待 ComfyuiProxy 支持后接入
- [ ] 成功后音频文件自动替换到角色资产

## Epic 3：分镜工作台

### US-007：AI 自动拆分章节为分镜脚本

**优先级**：Must Have

作为导演，
我希望点击"提取分镜"后 AI 自动从章节内容拆生成专业分镜脚本，
以便减少手动拆分分镜的工作量。

**验收标准**：
- [ ]  Production 界面有"提取分镜"按钮
- [ ] 点击后 AI 分析章节内容并拆分
- [ ] 每个分镜包含：镜头编号、描述、帧（首帧/中帧/末帧）
- [ ] 解析失败的章节回退到演示数据
- [ ] 分镜数据显示在右侧分镜面板

### US-008：AI 生成分镜图

**优先级**：Must Have

作为导演/美术，
我希望每个分镜可以调用 AI 生成对应的分镜图，
以便直观看到分镜效果。

**验收标准**：
- [ ] 每个分镜有"生成分镜图"按钮
- [ ] 点击后显示"生成中..."状态
- [ ] 成功后图片显示在分镜图位置
- [ ] 超时返回 promptId，手动"获取资产"

### US-009：AI 生成首帧图

**优先级**：Must Have

作为导演/美术，
我希望每个分镜的首帧可以调用 AI 生成高质量的起始画面，
以便为视频生成提供基础素材。

**验收标准**：
- [ ] 每个分镜有"首帧图"生成按钮
- [ ] 点击后显示"生成中..."状态
- [ ] 成功后图片显示在首帧位置
- [ ] 支持点击预览大图

### US-010：AI 生成分镜视频

**优先级**：Must Have

作为导演，
我希望每个分镜可以调用 AI 将首帧图生成分镜视频，
以便预览动态效果。

**验收标准**：
- [ ] 每个分镜有"生成视频"按钮
- [ ] 点击后需有首帧图（无首帧图则提示先生成）
- [ ] 点击后显示"生成中..."状态
- [ ] 成功后视频 URL 存储到分镜状态
- [ ] 超时返回 promptId，手动"获取资产"

## Epic 4：通用交互

### US-011：异步任务超时后的资产获取

**优先级**：Must Have

作为任何创作者，
我希望当 AI 任务超时时可以点击"获取资产"按钮继续查询结果，
以便不因超时而丢失已生成的资产。

**验收标准**：
- [ ] 超时后前端提示："生成中，请点击'获取资产'"
- [ ] 按钮变更为"获取资产"和"重新生成"
- [ ] 点击"获取资产"后能获取到已生成的结果
- [ ] 轮询间隔为 5 秒
- [ ] 超时时间为 10 分钟

### US-012：ComfyUI 不可用时的错误提示

**优先级**：Must Have

作为任何创作者，
我希望当 ComfyUI 或代理不可用时看到明确的错误提示，
以便知道问题原因并选择后续操作。

**验收标准**：
- [ ] 代理不可达时前端显示"代理服务连接失败"
- [ ] ComfyUI 服务器不可用时显示"AI 服务不可用"
- [ ] 错误提示包含重试/取消按钮
