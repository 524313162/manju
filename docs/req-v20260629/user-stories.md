# 漫剧开发工具 用户故事文档

> 版本：1.0 | 日期：2026-06-29 | 关联需求文档：requirements.md

## 用户角色

| 角色 | 描述 |
|------|------|
| 漫剧创作者 | 使用 ComfyUI 制作漫剧的个人或小团队成员，本地 PC 操作 |

---

## Epic 1：项目管理

### US-001：创建漫剧项目

**优先级**：Must Have

作为漫剧创作者，
我希望创建一个新的漫剧项目，
以便开始组织我的创作内容。

**验收标准**：

- [ ] 用户可以输入项目名称创建项目
- [ ] 项目创建成功后自动跳转到项目详情页面
- [ ] 项目名称具有唯一性约束

### US-002：管理漫剧项目列表

**优先级**：Must Have

作为漫剧创作者，
我希望查看、编辑和删除已完成或草稿项目，
以便管理多个漫剧作品。

**验收标准**：

- [ ] 首页显示所有项目列表
- [ ] 用户可以编辑项目名称
- [ ] 用户可以删除项目（含级联删除关联数据）

**优先级**：Must Have

作为漫剧创作者，
我希望系统自动生成演员的四视图提示词，

**优先级**：Must Have

作为漫剧创作者，
我希望上传自定义图片替换 ComfyUI 生成的演员四视图，

**优先级**：Must Have

- [ ] 用户可以删除分镜

### US-017：拖拽排序分镜

**优先级**：Must Have

作为漫剧创作者，

**验收标准**：

- [ ] 分镜列表支持拖拽交互
- [ ] 拖拽后顺序实时更新并保存

### US-018：替换首帧图和视频

**优先级**：Must Have

- [ ] 上传内容实时更新显示

## Epic 11：素材生成

### US-020：生成图片素材

**优先级**：Must Have

作为漫剧创作者，
我希望通过 ComfyUI 批量生成演员/道具/场景/首帧图片，
以便获取制作素材。

**验收标准**：

- [ ] 用户可以触发单个或批量图片生成
- [ ] 生成过程显示加载状态
- [ ] 生成完成后显示结果

### US-021：生成视频素材

**优先级**：Must Have

作为漫剧创作者，
我希望通过 ComfyUI 用首帧图+提示词生成镜头视频，
以便获得最终视频素材。

**验收标准**：

- [ ] 用户可以触发单个或批量视频生成
- [ ] 生成过程显示加载状态
- [ ] 提供获取资源按钮，用户可主动点击查看结果

### US-022：视频合并导出

**优先级**：Must Have

作为漫剧创作者，
我希望将所有镜头视频按顺序合并为完整作品，
以便导出最终成品。

**验收标准**：

- [ ] 用户可以点击合并导出按钮
- [ ] 系统校验 FFmpeg 是否可用
- [ ] FFmpeg 不可用时提示用户安装
- [ ] 合并完成后提示导出成功并提供下载
